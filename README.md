# FocusFlow - Productivity Workflow

FocusFlow, çalışan verimliliğini değerlendirmeye yönelik basit bir süreç yönetimi ve analiz uygulamasıdır. Proje, yapay zekâ tabanlı göz/odak takibi sonucunu simüle ederek kullanıcıya verimlilik değerlendirmesi sunmayı amaçlar.

## Project Purpose

Bu projenin amacı, çalışanların odaklanma, dikkat ve iş akışı verilerini analiz ederek verimlilik sürecini daha anlaşılır hale getirmektir.

## Features

- Ana sayfa
- Verimlilik değerlendirme ekranı
- Göz/odak takibi sonucu simülasyonu
- Basit analiz sonucu gösterimi
- Süreç yönetimi odaklı arayüz

## Technologies

- Next.js
- TypeScript
- JavaScript
- CSS
- Vercel Deployment

## Current Status

Projenin temel yapısı oluşturulmuştur. Ana sayfa üzerinde çalışmalar yapılmış olup, sonraki aşamada analiz ekranı ve kullanıcı akışı geliştirilecektir.
